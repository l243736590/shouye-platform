# 售业微信小程序

这是售业的原生微信小程序第一版，独立放在 `miniapp/`，不影响现有 Vite + React 网页部署。

## 当前功能

- 首页：灰色主题分流入口，先让用户选择“我要分享/助人”或“我要提问/求助”。
- 问答：读取 `https://shouye.fun/api/questions`，用小红书式两列图文流展示求助内容，并穿插商家软广。
- 解决：给助人/分享用户看的悬赏求助信息流，卡片显示提现积分和点赞。
- 经验：读取 `https://shouye.fun/api/posts`，以信息流区分经验帖和技能服务。
- 学校：学校专题入口，按问题类型引导，点击学校复制网页版专题链接。
- 商家：先选择留学咨询、学业相关、餐饮、物流、通信、家政搬家、不动产等分类，再以图文流展示商家。
- 发布：支持提问/悬赏、我知道、我能做三种发布入口，登录后复用线上 API。
- 我的：邮箱登录、查看消费积分/可提现积分、快速发布。

## 本地打开

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择：`E:\codex-workspaces\shouye-platform\miniapp`
4. AppID 可先用测试号/游客模式；正式发布前把 `project.config.json` 里的 `appid` 换成真实小程序 AppID。
5. 为了本地预览方便，`project.config.json` 里暂时关闭了 `urlCheck`；正式联调前建议打开，并在小程序后台配置合法域名。

## 正式上线前必须配置

在微信公众平台的小程序后台配置合法域名：

- request 合法域名：`https://shouye.fun`
- downloadFile 合法域名：`https://shouye.fun`（后续如果下载材料需要）
- uploadFile 合法域名：后续若做图片/材料上传再添加

如果要开放发布、评论、悬赏接单等 UGC 能力，正式审核前还需要补充内容安全审核、投诉入口、用户协议和隐私政策。
